## Introduction

The XCM-PDF Editor is a professional-grade tool developed by XcaliburMoon Web Development specifically for educational institutions and organizations. This application allows you to:

- View and navigate PDF documents
- Add annotations and comments
- Insert digital signatures with cryptographic security
- Fill out and complete forms
- Merge multiple PDF documents
- Export and share modified documents

### Learning Objectives

By completing this training, you will be able to:
- Navigate and operate the PDF editor interface
- Use all annotation tools effectively
- Create and apply digital signatures
- Complete PDF forms accurately
- Merge and organize PDF documents
- Save and export your work properly

## Setup Instructions

### Step 1: Installation

**For Windows users:**
1. Navigate to the pdf-editor folder
2. Double-click `install.bat`
3. Wait for installation to complete
4. Press any key to close the window

**For Mac/Linux users:**
1. Open Terminal
2. Navigate to the pdf-editor folder
3. Run: `./install.sh`
4. Wait for installation to complete

### Step 2: Verify Installation

After installation completes, you should see a message confirming success.

### Step 3: Generate Example PDF

**For Windows users:**
1. Double-click `generate-pdf.bat`
2. Wait for PDF generation
3. Press any key to close

**For Mac/Linux users:**
1. Open Terminal
2. Run: `./generate-pdf.sh`
3. Wait for completion

### Step 4: Launch the Application

**For Windows users:**
1. Double-click `run.bat`
2. Application will open in your browser

**For Mac/Linux users:**
1. Open Terminal
2. Run: `./run.sh`
3. Application will open in your browser

## Training Exercises

### Exercise 1: Opening and Navigating PDFs (10 minutes)

**Objective:** Learn to open and navigate through PDF documents

1. Click the "Open PDF" button
2. Select `public/example-document.pdf`
3. Wait for the document to load
4. Observe the page count in the sidebar
5. Practice navigating:
   - Click thumbnails in sidebar to jump to pages
   - Use zoom controls to zoom in/out
   - Scroll through the document

**Success Criteria:**
- Can open a PDF file
- Can navigate to specific pages
- Can zoom in and out effectively

### Exercise 2: Adding Text Annotations (15 minutes)

**Objective:** Learn to add and edit text annotations

1. Navigate to Page 3 of the example document
2. Click the Text tool (T icon) in the toolbar
3. Click in the "TEST AREA FOR ANNOTATIONS" box
4. Type: "This is my first annotation"
5. Practice moving the text:
   - Click Select tool (cursor icon)
   - Click on your text
   - Drag to new position
6. Add more text annotations with different content

**Success Criteria:**
- Can activate the text tool
- Can add text to the document
- Can reposition text annotations
- Can select and modify existing text

### Exercise 3: Digital Signatures (20 minutes)

**Objective:** Learn to create and apply digital signatures

1. Navigate to Page 4 of the example document
2. Click the Signature tool (pen icon) in the toolbar
3. In the signature pad that appears:
   - Draw your signature using your mouse/trackpad
   - If you make a mistake, click "Clear"
   - When satisfied, click "Apply"
4. Position the signature in the signature box
5. Click to place the signature
6. Observe the cryptographic signature applied

**Success Criteria:**
- Can open the signature pad
- Can draw a signature
- Can place signature on document
- Understands signature is cryptographically secured

### Exercise 4: Form Completion (25 minutes)

**Objective:** Complete a full PDF form using various tools

Navigate to Page 4 and complete the "Educational Project Participation Form":

1. **Fill in text fields:**
   - Use Text tool to add your name
   - Add current date (or use Date tool)
   - Add project name: "Leto's Angels Educational Project"
   - Add your institution/organization

2. **Check boxes:**
   - Use Checkbox tool
   - Click checkboxes for your areas of interest
   - Verify checkboxes appear correctly

3. **Add signature:**
   - Use Signature tool
   - Sign in the signature box

4. **Review your work:**
   - Use Select tool to adjust any elements
   - Ensure all fields are completed

**Success Criteria:**
- All text fields completed
- Checkboxes selected appropriately
- Signature applied correctly
- Form is neat and professional

### Exercise 5: Highlighting and Annotations (15 minutes)

**Objective:** Learn to highlight important content

1. Navigate to Page 1
2. Click the Highlight tool
3. Highlight the title "XCM-PDF EDITOR"
4. Navigate to Page 2
5. Highlight at least 3 key features
6. Use different colors if available

**Success Criteria:**
- Can activate highlight tool
- Can highlight text sections
- Can apply highlights to multiple pages

### Exercise 6: Adding Images (Advanced - 15 minutes)

**Objective:** Learn to insert images into PDFs

1. Find or create a small image file (logo, photo, etc.)
2. Navigate to Page 3
3. Click the Image tool
4. Select your image file
5. Position the image in the test area
6. Resize if needed

**Success Criteria:**
- Can select and insert image
- Can position image on document
- Can resize image appropriately

### Exercise 7: Saving Your Work (10 minutes)

**Objective:** Learn to save modified PDFs

1. Review all your annotations and edits
2. Click the "Save" button in the header
3. Choose a save location
4. Name your file: "YourName-Practice-Document.pdf"
5. Save the file
6. Locate the saved file on your computer
7. Open it to verify all changes were saved

**Success Criteria:**
- Can access save function
- Can choose save location and filename
- Saved PDF contains all modifications
- Can locate and open saved file

### Exercise 8: Merging PDFs (Advanced - 20 minutes)

**Objective:** Learn to merge multiple PDF documents

1. Create or locate 2-3 different PDF files
2. Click the "Merge" button
3. Add your PDF files to the merge list
4. Arrange them in desired order
5. Click "Merge" to combine
6. Save the merged document

**Success Criteria:**
- Can open merge interface
- Can add multiple PDFs
- Can arrange PDF order
- Can save merged document

## Real-World Applications

### Use Case 1: Educational Assignments

**Scenario:** A teacher needs to annotate student work with feedback

**Steps:**
1. Open student's submitted PDF
2. Use Highlight tool to mark important sections
3. Use Text tool to add comments and feedback
4. Use Signature tool to sign the review
5. Save and return to student

### Use Case 2: Administrative Forms

**Scenario:** Completing enrollment or permission forms

**Steps:**
1. Open the form PDF
2. Fill in all required text fields
3. Check applicable boxes
4. Add date stamps
5. Add digital signature
6. Save completed form for submission

### Use Case 3: Document Collaboration

**Scenario:** Multiple reviewers need to comment on a document

**Steps:**
1. Open document for review
2. Add text annotations with comments
3. Highlight sections that need attention
4. Add signature to indicate review complete
5. Save with reviewer name in filename

### Use Case 4: Creating Course Materials

**Scenario:** Combining multiple documents into one resource

**Steps:**
1. Collect all PDF materials
2. Use Merge function to combine
3. Add cover page annotations
4. Add page numbers or dates
5. Save as complete course packet
